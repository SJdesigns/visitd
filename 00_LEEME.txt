🚀 CÓMO EMPEZAR EN 3 PASOS
═══════════════════════════════════════════════════════════════════════════════

PASO 1: Lee PRIMEROS_PASOS.txt (1 minuto)
   👉 Guía ultra simple de 6 pasos

PASO 2: Obtén credenciales de Supabase (3 minutos)
   👉 Ve a https://supabase.com
   👉 Crea proyecto
   👉 Obtén credenciales en Settings > API

PASO 3: Configura MetroVisit (2 minutos)
   👉 Abre login/index.html y index.html
   👉 Añade credenciales en el <head>

PASO 4: Crea base de datos (2 minutos)
   👉 Copia DATABASE_SETUP.sql
   👉 Ejecuta en Supabase SQL Editor

PASO 5: Inicia servidor (1 minuto)
   👉 Terminal: python -m http.server 8000
   👉 Browser: http://localhost:8000/login/

PASO 6: Login (1 minuto)
   👉 Usuario: testuser
   👉 Contraseña: password123

⏱️ TIEMPO TOTAL: ~10 minutos
